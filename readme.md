# Lógica Backend

Pequeño proyecto para practicar la lógica de backend con JavaScript.
Permite agregar, mostrar, actualizar y eliminar tareas en memoria.

## Cómo usar

1. Clona el repositorio:
   git clone https://github.com/aedh1285-coder/logica-backend.git

2. Abre el proyecto y ejecuta con Node.js:
   node indexrevlist.js

## Qué aprendí

- Manipulación de arrays y objetos
- Funciones CRUD
- Condicionales y loops
- Manejo de commits y GitHub
